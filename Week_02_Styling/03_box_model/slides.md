## Nesting Selectors

p {}

vs.

.myClass p {}

---

## Select Multiple Elements

Use a comma

.myClass, p, h2 {}

---

## Select element only inside another element

.myClass p {}

---

## Select only IMMEDIATE child of a particular element

.myclass > p {}

---

## Select exact match for both tag AND class combination

p.myclass {}
